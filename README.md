# Smart CLI Calculator

A Python-based Smart CLI Calculator developed by **Sanjay Jangid**.

## Features
- Basic Arithmetic Operations (+, -, *, /)
- Scientific Functions
  - Sin
  - Cos
  - Tan
  - Log
  - Square Root
  - Power
- Unit Conversion
  - cm ↔ inch
  - kg ↔ lb
  - Celsius ↔ Fahrenheit

## Technologies Used
- Python 3
- Math Module

## How to Run

```bash
python smart_cli_calculator.py
```
